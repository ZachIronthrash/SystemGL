This Visual Studio solution is a combination library and set of demonstrations for computational physical models.

Capable of simulating non-interacting particles in a box:
![anhrqf](https://github.com/user-attachments/assets/1c26080e-5a0e-4ed2-94d5-012cfe40627a)
![anhrux](https://github.com/user-attachments/assets/7255cfeb-5aac-487f-9ef9-c6e1d0360b37)

Which includes an accurate approximation of the pressure an ideal gas exerts (with no gravity):
![anhtho](https://github.com/user-attachments/assets/f6b630d3-db99-4cf4-bfac-d6b6b71d8f83)
![anhtq7](https://github.com/user-attachments/assets/6144e530-1336-4359-97ca-0c9ffd47d4c9)

Reference libraries are located in external.
